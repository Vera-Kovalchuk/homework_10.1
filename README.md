# Проект "Виджет банковских операций клиента"

## Описание:

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/username/project-x.git
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Требования к окружению:
```
Python 3.8+
```
